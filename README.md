- [**Fractal Glide**](https://store.steampowered.com/app/2565200/Fractal_Glide/) - A fractal game where the player glides through fractal landscapes (available on Steam!)

- [**Unity FractiX**](https://github.com/NabilNYMansour/Unity-FractiX) - An open-source cone marching/ray marching rendering engine written as a post-effect to the Unity rendering pipeline

- [**Slime Simulation**](https://github.com/NabilNYMansour/Slime-Simulation) - An implementation of slime mold cellular automata written in Python
- [**PyLisp**](https://github.com/NabilNYMansour/PyLisp) - A Lisp interpreter written in Python loosely following John McCarthy's [original Lisp paper](https://people.cs.umass.edu/~emery/classes/cmpsci691st/readings/PL/LISP.pdf)
- [**Self Parallel Parking Arduino Car**](https://github.com/NabilNYMansour/CPS607-Self-Parallel-Parking) - An Arduino car that parallel parks on its own
- [**Wireless Arduino RC Car**](https://github.com/NabilNYMansour/Wireless-RC-Car) - A wireless Arduino RC car made out of cardboard and controlled using radio antennas
- [**Voxel Terrain**](https://github.com/NabilNYMansour/Voxel-Terrain) - A Unity demo that showcases a randomly generated voxel terrain based on Minecraft
